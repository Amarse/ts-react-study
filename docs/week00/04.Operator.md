### 2. 비교 연산자
| operator  | example    | explain                        |
| :---------| :--------- |:------------------------------:|
| ==        | a == b     | a 와 b의 값이 같은지 비교        |
| ===       | a === b    | a 와 b의 값과 타입이 같은지 비교  |
| !=        | a != b     | a 와 b의 값이 다른지 비교        |
| !==       | a !== b    | a 와 b의 값과 타입이 다른지 비교 |
| >         | a > b      | a 가 b 보다 큰지 비교            |
| <         | a < b      | a 가 b 보다 작은지 비교          |
| >=        | a >= b     | a 가 b 보다 크거나 같은지 비교   |
| <=        | a <=       | a 가 b 보다 작거나 같은지 비교   |
